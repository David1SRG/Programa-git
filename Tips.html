多Como hacer el arroba en un portatil?

Para usar el arroba en un portatil usa las combinaciones de teclas PC: alt gr + Q

多Como cortar el texto seleccionado?

Para cortar, usa las combinaciones de teclas PC: ctrl + X

多Como deshacer?

多Como rehacer?

Para rehacer la accion que eliminaste, usa las combinaciones de teclas PC: ctrl + Y
